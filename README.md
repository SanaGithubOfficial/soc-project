# 🔍 Real-World Internship Scam Analysis

## 📌 Overview
This project presents a real-world cybersecurity investigation of a suspicious internship offer.

The analysis was performed using email (.eml), PDF content, and URL inspection to identify indicators of social engineering, phishing behavior, and potential financial exploitation.

---

## 🎯 Objective
To analyze a suspicious internship offer and determine its legitimacy using basic SOC (Security Operations Center) investigation techniques.

---

## 🔍 Scope of Analysis

- 📧 Email Analysis (.eml)
  - Header inspection
  - Domain mismatch detection
  - Tracking links and bulk email infrastructure

- 📄 PDF Analysis
  - Payment requirement identification
  - Social engineering techniques (baiting, delayed reward)
  - Embedded link analysis

- 🌐 URL Analysis
  - Tracking-based URL structure
  - Encoded parameters (obfuscation)
  - Redirection behavior
  - Brand mismatch (LinkedIn reference)

- 🌍 Domain Analysis
  - WHOIS lookup
  - Domain age and registration details
  - Hosting and infrastructure insights

---

## 🛠️ Tools Used

- VirusTotal  
- WHOIS Lookup  
- Manual Static Analysis  

---

## 🚨 Key Findings

- Upfront payment required before joining (₹1563)  
- Domain mismatch (Clinchsoft vs clinchbridge)  
- Use of bulk email infrastructure  
- Tracking links and tracking pixel present  
- Encoded URL hiding actual destination  
- Brand impersonation (LinkedIn reference)  
- No structured hiring process  
- Misleading salary and stipend claims  

---

## 🛡️ Recommended Actions

Based on the analysis, the following precautions are advised when encountering similar opportunities:

- Avoid interacting with suspicious emails, links, or attachments  
- Do not make any payment for job or internship offers  
- Verify company details and domain authenticity  
- Report suspicious activity on job platforms or official cybercrime portals  
- Preserve evidence (email, screenshots, URLs) for further investigation
